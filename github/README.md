# git命令
## git rebase
### 简单操作
```
git rebase -i commit_digest
```
squash commit压缩,pick commit保存</br>
git reflog,git reset ref_digest 可以撤销rebase </br>
## git reset
git reset 信息摘要码
不要加把这次修改的本地内容也清除的选项
## 生成公钥
